## Getting Started

### Requirements

- Node.js 18+
- pnpm (preferred) or npm/yarn

### Installation

```bash
pnpm install
# or
npm install
```

### Start Development Server

```bash
pnpm dev
# or
npm run dev
```

Open your browser at [http://localhost:3000](http://localhost:3000)

---

## Build for Production

```bash
pnpm build
pnpm start
```
